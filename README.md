### Setup
$ git clone git@github.com:Rekunch/php-project-lvl1.git

$ make install

## you can start games with 'make' comands:
$ make brain-games <br>
$ make brain even <br>
$ make game-calc <br>
$ make game-gcd <br>
$ make game-progression <br>
$ make game-prime <br>
